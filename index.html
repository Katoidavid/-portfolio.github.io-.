My Projects

Classification

The aim of this project was to develop an image classification model that accurately categorizes different objects or regions within an image based on their backscatter values. By analyzing the intensity of signals reflected from the image's surface, I sought to extract meaningful features that can be used to discriminate between various classes.

Network Analysis

The objective of this project was to employ ArcGIS Pro's network analysis tools to identify the closest facility and generate service areas for a given dataset. By utilizing these geospatial techniques we aimed to optimize location-based decisions, assess accessibility, and understand spatial relationships within the study area.

Suitability Analysis

The aim of this project is to identify optimal locations for new school construction through a rigorous suitability analysis. By evaluating various environmental, social, and infrastructural factors, I aimed to delineate areas that best meet the criteria for school siting. This analysis will inform decision-making, ensuring that new schools are established in locations that effectively serve the community and minimize potential negative impacts.

NDVI Calculation

The project utilized Google Earth Engine to calculate and analyze NDVI, providing valuable insights into vegetation health, distribution, and changes over time for [specific area or application]. This project aimed to demonstrate the potential of remote sensing and cloud computing for effective environmental monitoring and management in Makueni County.
